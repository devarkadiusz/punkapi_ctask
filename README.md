Downloading repository
----------------------

    git clone https://github.com/devarkadiusz/punkapi_ctask.git
    cd punkapi_ctask

Installing dependencies
-----------------------

    # Frontend
    npm install

    # Backend
    npm --prefix src/backend install

Running
--------

    # npm
    
    ## Frontend
    npm start

    ## Backend
    npm run server

    # or use nodemon for the backend
    cd src/backend && nodemon server.js
